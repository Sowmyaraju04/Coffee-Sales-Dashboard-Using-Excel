# Coffee Sales Dashboard (Excel Project)

## Overview
This project is an interactive **Coffee Sales Dashboard** built using Microsoft Excel.  
It provides key business insights into sales performance, customer behavior, and product trends through dynamic visualizations and filters.

---

## Objective
- Analyze coffee sales data to identify business trends  
- Track sales performance across countries, coffee types, and time periods  
- Understand top customers and product preferences  
- Build an interactive dashboard for decision-making  

---

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers & Timelines  
- Data Cleaning & Formatting  

---

##  Key Features
- Sales trend analysis over time  
- Country-wise sales breakdown  
- Coffee type performance comparison  
- Top customers insights  
- Interactive filters (Roast type, Size, Loyalty card, Date)  

---

## Dashboard Highlights
- Total sales performance tracking  
- Monthly and yearly sales trends  
- Product category analysis  
- Customer segmentation insights  

---



## How to Use
1. Open the Excel dashboard file  
2. Use slicers and timelines to filter data  
3. Explore charts for insights  
4. Analyze KPIs for decision-making  

---

##  Outcome
This dashboard helps transform raw sales data into meaningful insights, enabling better business decisions in retail coffee sales.
